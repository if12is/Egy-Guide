## To install Laravel

```
composer install  {composer update --ignore-platform-reqs }
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan g:c
php artisan g:s all _or_ php artisan g:s Egypt
npm install
npm run dev
php artisan serve

```
